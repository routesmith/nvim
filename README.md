# Neovim Configuration

Modular Neovim configuration based on kickstart.nvim with cross-platform support (Windows/WSL/macOS).

## Features
- 🚀 Lazy.nvim plugin manager
- 🔍 fzf-lua for fuzzy finding
- 💡 LSP with Mason
- ⚡ blink.cmp for completion
- 🎨 Multiple colorschemes (Gruvbox, Kanagawa, Tokyo Night)
- 🔧 Mini.nvim suite for lightweight utilities
- 🌐 Cross-platform clipboard & shell support

## Requirements
- Neovim >= 0.10
- Git
- fzf
- ripgrep (rg)
- fd (optional but recommended)
- bat (optional but recommended)
- **Windows**: win32yank.exe, PowerShell 7+
- **Unix**: xclip/xsel or pbcopy (macOS)
